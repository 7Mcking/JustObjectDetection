# JustObjectDetection
Object detection Gui using cascade classifier and DNN

## Required Libraries 
QT5, OpenCV

### Running the project
```git clone https://github.com/7Mcking/JustObjectDetection.git```

```mkdir build && cd build```

```cmake ..```

```make```

At the end run the executable
